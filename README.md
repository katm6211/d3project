# d3project
I'll have 3 physics-based mechanics implemented into my gameplay. These mechanics will be exemplified through the way the player uses the sprite character to interact with their environment. The collisions I'll be dealing with are the objects the sprite is around in their environment, the bounds of the canvas, and the output of interacting with objects in different ways. 
https://katm6211.github.io/d3project/ 
All of my assets were created by me using piskel.
